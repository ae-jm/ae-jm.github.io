텍스트 감정 분류 기반의 음악 추천 시스템

사진 클릭시 깃허브 링크로 이동!



팀 내 역할 : 기획 및 데이터 수집, 개발(1인 개발 프로젝트)

주요 활용 기술 : python, LSTM, KoNLPy, HTML, CSS, JavaScript, Django, Bootstrap5, MySQL

대전대학교 컴퓨터공학과 졸업 작품(캡스톤디자인) 프로젝트로, 텍스트 감정 분류 모델과 음악 추천 시스템을 결합하여 사용자가 작성한 일기를 분석하고 이를 토대로 사용자의 감정에 따른 음악을 추천해주는 일기 작성 웹 사이트를 개발하였다. 가사 데이터를 수집을 위해 Spyder를 사용하여 구글 크롬 웹 드라이버와 셀레니움을 이용해 웹 크롤링을 진행, 감정 분류 모델을 개발하기 위해 AI Hub의 감정 대화 말뭉치 데이터를 모델 학습에 사용하였다. 가사 데이터의 감정을 불안, 분노, 상처, 슬픔, 당황, 기쁨의 6가지 범주의 정량적 수치로 분류하여 유클리드 거리를 사용해 사용자가 작성한 일기의 감정과 가장 유사한 노래를 리스트업 하고 각각 레이더 차트를 제시하여 사용자가 눈으로 확인할 수 있게 구성하였다. 개발 IDE로 Pycharm을 사용하여 Django를 이용해 백엔드 개발을 하였고, Bootstrap5와 HTML, CSS, JavaScript를 사용해 프론트앤드 개발을 진행하였다.

OST를 이용한 영화 추천 웹사이트


팀 내 역할 : PM, 영화 추천 알고리즘 개발 및 front-end 참여

주요 활용 기술 : python, HTML, CSS, JavaScript, Django, Bootstrap5, MySQL

영화의 내용은 별로였지만 영화에 삽입된 음악만큼은 맘에 들었던 경험있나요? 이 프로젝트는 "영화의 OST는 영화의 전반적인 분위기를 대변해줄것이다." 라는 가정으로 시작된 프로젝트로, 영화 추천 알고리즘에 기본적으로 사용되는 feature들에 spotify에서 제공하는 음악의 정량적 feature를 더하여 추천 알고리즘을 만들어보았다. 장르와 감독 feature를 라벨링하여 spotify의 valence, acousticness, danceability, energy, loudness, tempo의 정량적 수치들과 함께 KMeans로 군집화를 진행하고 유클리드 거리를 계산하여 검색한 영화와 가장 유사한 영화를 리스트업 하는 사이트이다. 어떻게 유사한지 레이더 차트를 제시하여 사용자가 눈으로 확인할 수 있게 구성하였다. 개발 IDE로 Pycharm을 사용하여 Django를 이용해 백엔드 개발을 하였고, Bootstrap5와 HTML, CSS, JavaScript를 사용해 프론트앤드 개발을 진행하였다.

수화 번역 시스템


팀 내 역할 : 데이터 수집 및 전처리, 수화 번역을 위한 모델 개발 참여

주요 활용 기술 : python, LSTM, OpenCV, MediaPipe, HTML, CSS, JavaScript, Django, Bootstrap5, MySQL

아르바이트를 하던 시절, 언어장애를 가진 손님들과의 의사소통에 어려움을 겪었다. 이후 상용화 된 서비스 중에 텍스트를 수화동작으로 변환해주는 것은 있어도 수화 동작을 텍스트로 번역해주는 것은 없다는 것을 알게 되었다. 이 웹 사이트는 OpenCV와 MediaPipe를 이용하여 개발자가 수화 동작을 직접 수행해 데이터를 수집하였고, 모델 학습을 통해 사용자의 웹 상의 수화 동작을 실시간으로 번역해주는 서비스를 제공한다. 또한 이를 통해 수화 학습 도구로도 활용할 수 있으며 이외에도 장애인, 비장애인 상관없이 서로 소통할 수 있는 커뮤니티 창구로서의 역할 역시 수행될 수 있다. 개발 IDE로 Pycharm을 사용하여 Django를 이용해 백엔드 개발을 하였고, Bootstrap5와 HTML, CSS, JavaScript를 사용해 프론트앤드 개발을 진행하였다.
